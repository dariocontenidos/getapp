¿Cómo configurar Firebase? ›
1. Ir a console.firebase.google.com
2. Crear proyecto → Realtime Database → Crear base de datos
3. Reglas → editar: ".read": true, ".write": true
4. Engranaje → Config del proyecto → SDK → copiar JSON
5. Pegar el JSON arriba y elegir tu código personal
